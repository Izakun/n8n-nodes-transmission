# n8n-nodes-transmission

[![npm version](https://img.shields.io/npm/v/n8n-nodes-transmission.svg)](https://www.npmjs.com/package/n8n-nodes-transmission)

n8n community node for [Transmission](https://transmissionbt.com/) — the BitTorrent client — via its RPC API.

Install via **Settings -> Community Nodes -> Install** -> `n8n-nodes-transmission`.

## Operations
- Get Torrents, Get Session, Get Session Stats

## Credentials
Configure the base URL and authentication in the **Transmission API** credential.
